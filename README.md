# CloudSec Auditor

CloudSec Auditor is a Python and Boto3-based AWS security auditing tool.

The goal is to automate common cloud security checks including:

- S3 security
- IAM security
- Security Groups
- GuardDuty
- CloudTrail
- Encryption controls
